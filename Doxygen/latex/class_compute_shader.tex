\doxysubsection{Detailed Description}
\doxylink{class_compute_shader}{Compute\+Shader} class. 

This class includes the method definitions for creating, binding, and unbinding compute shaders within the \doxylink{class_compute_shader}{Compute\+Shader} class.

The \doxylink{class_compute_shader}{Compute\+Shader} class is used to create, bind, and unbind compute shaders. The class also includes methods for uploading data to the GPU, setting uniforms, and retrieving data from the GPU.
