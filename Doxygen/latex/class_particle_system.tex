\doxysubsection{Detailed Description}
Manages a system of particles, including creation, updating, and destruction. 

The \doxylink{class_particle_system}{Particle\+System} class manages a collection of particles in the simulation. It provides functionality for creating new particles, updating their properties, and destroying them. The class also maintains a list of unique identifiers for each particle in the system.
