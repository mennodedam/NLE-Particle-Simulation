\doxysubsection{Detailed Description}
\doxylink{class_renderer}{Renderer} class. 

This class includes the method definitions for creating, binding, and unbinding compute shaders within the \doxylink{class_compute_shader}{Compute\+Shader} class.

The \doxylink{class_compute_shader}{Compute\+Shader} class is used to bind and unbind shaders. The class also includes the method for drawing the vertex array.
