\doxysubsection{Detailed Description}
\doxylink{class_shader}{Shader} class. 

This class includes the method definitions for creating, binding, and unbinding shaders within the \doxylink{class_shader}{Shader} class.

The \doxylink{class_shader}{Shader} class is used to create, bind, and unbind shaders. The class also includes methods for uploading data to the GPU, setting uniforms, and retrieving data from the GPU.
